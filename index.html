Project Insights üìù

üìå HTML Element Used: <table>

For this project, I explored the <table> element, which is used to display structured data in a tabular format. The <table> element organizes content into rows and columns, making it ideal for schedules, financial data, and reports.

What I learned:

  • <thead>, <tbody>, and <tr> improve accessibility and readability.
  • <th> defines table headers, making them bold and centered by default.
  • You can span multiple rows/columns using rowspan and colspan.

üé® CSS Property Used: Smooth Transitions

For my CSS property experiment, I explored the transition property to create a smooth hover effect on images. The transition property allows for gradual changes to CSS properties over time, making animations feel natural.

How I used it:

  • When hovering over an image, it grows slightly (scale(1.1)) and fades (opacity: 0.9) to create a subtle effect.
  • This effect makes the images feel more interactive and visually engaging.
